{\bfseries{A cross-\/platform base for native Nickvision applications.}}

libnick provides Nickvision apps with a common set of cross-\/platform APIs for managing system and desktop app functionality such as network management, taskbar icons, translations, app updates, and more.\hypertarget{index_autotoc_md6}{}\doxysection{\texorpdfstring{Dependencies}{Dependencies}}\label{index_autotoc_md6}
The following are a list of dependencies used by libnick.\hypertarget{index_autotoc_md7}{}\doxysubsection{\texorpdfstring{All Platforms}{All Platforms}}\label{index_autotoc_md7}

\begin{DoxyItemize}
\item gtest
\item jsoncpp
\item libcurl
\item libintl
\item maddy
\end{DoxyItemize}\hypertarget{index_autotoc_md8}{}\doxysubsection{\texorpdfstring{Windows}{Windows}}\label{index_autotoc_md8}
The above dependencies must be installed, {\itshape plus} the following for Windows systems\+:
\begin{DoxyItemize}
\item sqlcipher
\end{DoxyItemize}\hypertarget{index_autotoc_md9}{}\doxysubsection{\texorpdfstring{Linux}{Linux}}\label{index_autotoc_md9}
The above dependencies must be installed, {\itshape plus} the following for Linux systems\+:
\begin{DoxyItemize}
\item glib
\item libsecret
\item libuuid
\item openssl
\begin{DoxyItemize}
\item Used for sqlcipher, as libnick manually builds sqlcipher on Linux as the vcpkg port is broken.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{index_autotoc_md10}{}\doxysubsection{\texorpdfstring{mac\+OS}{mac\+OS}}\label{index_autotoc_md10}
The above dependencies must be installed, {\itshape plus} the following for mac\+OS systems\+:
\begin{DoxyItemize}
\item glib
\item libsecret
\item openssl
\begin{DoxyItemize}
\item Used for sqlcipher, as libnick manually builds sqlcipher on mac\+OS as the vcpkg port is broken.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{index_autotoc_md11}{}\doxysection{\texorpdfstring{Consuming libnick via vcpkg}{Consuming libnick via vcpkg}}\label{index_autotoc_md11}
libnick is available through {\ttfamily vcpkg}.

Simply install and configure vcpkg for your system, and run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{vcpkg\ install\ libnick}

\end{DoxyCode}
 You can then use libnick from cmake with the following\+: 
\begin{DoxyCode}{0}

\DoxyCodeLine{target\_link\_libraries(main\ PRIVATE\ libnick::libnick)}

\end{DoxyCode}
\hypertarget{index_autotoc_md12}{}\doxysection{\texorpdfstring{Building and Installing Manually}{Building and Installing Manually}}\label{index_autotoc_md12}
libnick uses {\ttfamily vcpkg} to manage its dependencies and {\ttfamily cmake} as its build system.

Ensure both {\ttfamily vcpkg} and {\ttfamily cmake} are installed on your system before building and installing libnick.
